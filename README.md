What is this?
=============
This is a simple app designed to provide offline access to RSS feeds. this app won't access any special dependancies and will be completely self contained.

Testing Instructions
====================

1. Download the latest build of Node-Webkit: [https://github.com/rogerwang/node-webkit][nodewebkit] ([direct link][nodewebkitdirect]).
2. Extract it to the `node-webkit` folder.
3. Run `test.bat`
